## Usage

```tsx
import { Tabs } from '@aussieljk/frosted';

<Tabs.Root defaultValue="account">
  <Tabs.List>
    <Tabs.Trigger value="account">Account</Tabs.Trigger>
    <Tabs.Trigger value="settings">Settings</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="account">…</Tabs.Content>
  <Tabs.Content value="settings">…</Tabs.Content>
</Tabs.Root>;
```

## Parts

| Part           | Purpose                                             |
| -------------- | --------------------------------------------------- |
| `Tabs.Root`    | State container — `value`, `defaultValue`, `onValueChange` |
| `Tabs.List`    | Row of triggers                                     |
| `Tabs.Trigger` | Activates a tab — `value`                           |
| `Tabs.Content` | Panel shown for the matching `value`                |

Related: `TabsNav` for link-based (routing) tab bars and `SegmentedControl` for a filled toggle
group.

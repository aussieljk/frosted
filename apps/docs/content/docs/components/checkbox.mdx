## Usage

```tsx
import { Checkbox, Text } from '@aussieljk/frosted';

<Text render={<label />} size="2">
  <Checkbox defaultChecked /> Agree to terms
</Text>;
```

Supports controlled usage via `checked` / `onCheckedChange`, plus `defaultChecked` and
`disabled`.
